========
概要
========

その名のとおり、画面を「Happy New Year」で覆い尽くすプリロードです。

ある意味、ジョークな機能を追加できるプリロードで、「ばばんと画像が登場するプリロード」の改造版です。
http://xoops123.com/modules/d3downloads/index.php?page=singlefile&cid=3&lid=62

プリロードを入れると、「ばば～ん」と「Happy New Year」画像が登場して、画面を覆い尽くします。
そして、それをクリックすると本来の画面に戻るというもので、新年の雰囲気をアップするギミックとしてご利用いただければと思います～(^^ゞ

サンプルとして、XOOPS Cube 日本サイトのデザイン画像に新年らしさをプラスした画像を表示するようにしています。

Ver0.02から、動作開始する日付を指定して、その日から動作するようにしてみました。
初期設定として、2013年１月１日午前０時に設定していますが、変更する場合は、３１行目の
　　var endTime = new Date(\'January 1,2013, 00:00:00\');
を変更してください。

また、初期状態では、毎回各ページにアクセスするたびに画像を表示するようになっていますが、プリロード HappyNewYear.class.php ファイル内の４９行目
 　jQuery.cookie("access_site","you", { expires: -1 });
を次の記述のように
　 //jQuery.cookie("access_site","you", { expires: -1 });
とコメントアウトするとクライアント側のブラウザにクッキーを保存して、一定期間表示しないような機能を付加しています。（初期設定では、３６４日。変更は、プリロードファイル内を修正してください）

よろしければご利用くださいませ。
